# 家校日历小程序 - 项目方案

## 一、功能概述

**目标用户：** 学校家委会成员、家长、老师

**核心功能：**
1. 发布活动（文字/海报图片）
2. 日历查看活动
3. 活动提醒（微信推送）

---

## 二、功能详情

### 1. 活动发布（老师/家委会）
- **文字活动**：标题、时间、地点、描述
- **海报活动**：上传图片（海报）+ 文字说明
- **发布平台**：小程序管理后台 或 直接微信群转发

### 2. 日历展示（家长）
- **月视图**：查看每月活动
- **列表视图**：近期活动列表
- **活动详情**：点击查看完整信息（包括海报图片）

### 3. 提醒通知
- **当天提醒**：活动开始前推送
- **自定义提醒**：可选1小时/1天前提醒

---

## 三、技术方案

### 微信小程序
- **前端**：微信小程序原生开发 或 Uni-app
- **后端**：微信云开发（云数据库 + 云函数）

### 数据结构
```
Event {
  id: string,
  title: string,
  description: string,
  imageUrl: string,  // 海报图片
  dateTime: datetime,
  location: string,
  remindBefore: number,  // 提醒时间（分钟）
  createdBy: string,
  createdAt: datetime
}
```

### 通知方案
- 使用微信订阅消息（Subscribe Message）
- 定时触发云函数检查活动提醒
